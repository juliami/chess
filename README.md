# Fantasy Chess (React Native + Reanimated)

A high-performance, fantasy-themed chess app demonstrating **React Native Reanimated 3** and **Gesture Handler**.

![React Native](https://img.shields.io/badge/React_Native-blue) ![Reanimated](https://img.shields.io/badge/Reanimated-v3-purple) ![TypeScript](https://img.shields.io/badge/TypeScript-blue)

## Project Goal

Build drag & drop chess game with React Native and Reanimeted. Made for educational purposes.



## Stack

* **React Native** (Expo)
* **React Reanimated**
* **Gesture Handler**
* **js-chess-engine**

## Running locally

```bash
npm install
npx expo start
```
